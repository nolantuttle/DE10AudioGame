Compilation Instructions:

To compile the audioGame program on a Linux/UNIX operating system, use the following command in this directory's command line interface:

make

This will generate an executable program in the /bin/ directory.

Running the Program:

To run the executable program that was generated, run the following command:

./bin/audioGame

To run the file after making changes, run the following commands:

make clean

make
